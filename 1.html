HTML

Hyper Text Markup Language (HTML) is the standard markup language forcreating web page and web applications

HTML5 is the fifth generation and the latest specification of the HTML.

It is a newer version of the HTML, and has support for multimedia, structural elements, and promotes accessibility and responsiveness to allow device-independent markup. HTML5 encourages the separation of content from style, and deprecated styling elements to encourage the use of the CSS.
